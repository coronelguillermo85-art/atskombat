**ATS KOMBAT ULTRA** es la cuarta entrega de mi ecosistema de postulación gamificada. En este proyecto, elevo la complejidad técnica pasando de un juego de plataformas a un **simulador de combate 2D** con una IA reactiva. Es la demostración final de cómo mi perfil híbrido en **Soporte Técnico**, **RR.PP.** e **Innovación Pública (UNER)** puede transformar un proceso administrativo en una experiencia disruptiva.

## 🕹️ Mecánicas de Combate (Gameplay)

El juego simula una pelea arcade clásica de 16 bits con un set de movimientos técnicos:
* **Z - Punch (CV HIT):** Ataque rápido de corto alcance para generar presión.
* **C - Kick (STACK KICK):** Ataque de mayor daño y rango.
* **X - Special (CV ATTACK):** Proyectil de largo alcance que representa el impacto directo del perfil técnico.
* **IA Adaptativa:** El **ATS Bot** analiza la distancia del jugador para decidir entre retroceder, atacar cuerpo a cuerpo o lanzar filtros de keywords.

## 🛠️ Stack Técnico e Ingeniería

Este repositorio es una auditoría de código en tiempo real que demuestra habilidades en:

* **Custom Physics Engine:** Gestión manual de gravedad, fricción y vectores de movimiento sin motores externos.
* **Hitbox System:** Detección de colisiones precisa mediante intersección de rectángulos para ataques de puño, patada y proyectiles.
* **State Machine:** Control de estados del luchador (Hitstun, Invulnerabilidad, Animación de ataque) para garantizar un gameplay fluido a 60fps.
* **Visual FX:** Sistema de partículas dinámicas para impactos y "Screen Shake" para dar peso a la acción.

## 🏛️ El Enfoque Estratégico

El juego culmina con dos estados posibles:
1.  **HIRED (Victoria):** Demuestra que el candidato superó las barreras tecnológicas. Se desbloquea el acceso al CV real.
2.  **REJECTED (Derrota):** El sistema ganó esta ronda, invitando al usuario a "reintentar y contratar mejor".

Es una metáfora de la **Gestión del Cambio**: la tecnología no debe ser una barrera, sino una herramienta de conexión entre el talento y la oportunidad.
